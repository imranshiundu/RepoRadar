# Repo Radar ⬡

Repo Radar is a high-performance developer intelligence dashboard designed to scan the horizon for the most promising open-source projects, SaaS opportunities, and viral tech trends. It aggregates signals from across the web—including GitHub Trending, Reddit, Hacker News, and Product Hunt—to provide a unified view of what's happening in the engineering world right now.

![Repo Radar Dashboard](https://raw.githubusercontent.com/imranshiundu/RepoRadar/main/screenshot.png) *(Placeholder for actual screenshot)*

## 🚀 Key Features

- **Multi-Source Aggregation**: Unified feed from GitHub, r/programming, r/SideProject, r/MachineLearning, Hacker News, Product Hunt, and more.
- **AI-Powered Intelligence**: Integrates with Groq (Llama 3/Mixtral) to provide:
    - **Auto-Summarization**: Instant, high-level summaries of complex repositories.
    - **Monetization Scoring**: Identifies projects with high potential for SaaS conversion or commercial use.
    - **Tech Stack Detection**: Automatically detects and tags the underlying technologies.
- **Smart Folders**: Intelligent categorization into folders like "AI + LLM Tools", "SaaS Starters", "CLI Wizardry", and "Ship This Weekend".
- **Personalized Discovery**: Tailor your feed based on specific interests and keywords.
- **Real-Time Signals**: Live alerts for star milestones, "Show HN" drops, and language trends.
- **Premium UI/UX**: A sleek, dark-mode-first interface built for speed and clarity, featuring glassmorphism and dynamic micro-animations.

## 🛠 Getting Started

### Prerequisites
- A modern web browser.
- (Optional) A [Groq API Key](https://console.groq.com/) for AI-powered features.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/imranshiundu/RepoRadar.git
   ```
2. Open `index.html` in your browser.
3. (Optional) Click on **⚙ Config** and enter your Groq API key to enable AI analysis.

## ⚙ Configuration
Repo Radar is highly configurable. Open the **Config** panel to:
- Toggle data sources on/off.
- Customize AI analysis depth.
- Set your specific interests for personalized recommendations.
- Manage API tokens for increased rate limits.
